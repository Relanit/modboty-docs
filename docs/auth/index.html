Авторизация

Twitch требует у создателей ботов запрашивать у пользователей прохождение авторизации для получения доступа к некоторым функциям.

Список функций:

  • Управление випами
  • Опросы
  • Ставки
  • Настройки стрима

Для прохождения авторизации, перейдите по ссылке, это может занять до минуты.
