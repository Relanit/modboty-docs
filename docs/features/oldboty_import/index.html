Импорт команд OldBoty

Импорт и удаление команд из OldBoty.

Общая информация
  • Название: oldimport
  • Элиасы: olddel
  • Кулдаун: общий 5 секунд
  • Исходный код

Импорт команд​

!oldimport

Удаление команд из OldBoty после импорта​

!olddel
