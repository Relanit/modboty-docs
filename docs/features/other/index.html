Дополнительные функции

Другие полезные функции

Помощь​

!help ‒ список команд

!help value

  • value ‒ название или элиас команды
к сведению

С помощью !help можно узнать информацию о любой команде бота, элиасах категорий, а также о кастомных командах , их элиасах и таймерах

Спам текстом​

!spam number value

  • number ‒ сколько раз отправить текст
  • value ‒ текст для спама

Пример:

!spam 10 Текст для спама

Проверка бота​

!ping
