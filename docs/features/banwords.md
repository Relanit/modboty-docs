# Банворы/мутворды

Запрещённые слова, за отправку которых пользователь получает бан/мут

## Создание банворда
`!bword value`
- `value` - фраза

## Создание мутворда
`!mword duration value`
- `duration` - длительность мута в секундах
- `value` - фраза

## Просмотр банвордов/мутвордов
- `!bwords`
- `!mwords`

## Удаление банвордов/мутвордов
Банворды - `!delb value`

Мутворды - `!delm value`
