Автообновление оверлея jChat

Обновление оверлея при изменении смайлов на стриме

Если вы пользуетесь оверлеем jChat, вы можете включить его автообновление, тогда во время стрима бот будет автоматически отправлять команду !refreshoverlay при добавлении/удалении смайлов

Общая информация
  • Название: refreshchat
  • Элиасы: отсутствуют
  • Кулдаун: общий 5 секунд
  • Исходный код

Включение автообновления​

!refreshchat value

  • value ‒ on или off для включения или выключения соответственно
