# Разрешения
Бот не будет реагировать на пользователя в течение указанного времени

`!allow user time`
- `user` ‒ ник пользователя
- `time` ‒ время в секундах (по умолчанию 180)


    !allow relanit 300

## Отмена разрешения
`!disallow user`


    !disallow relanit

## Приостановка работы антифлуда
`!allow time`
- `time` ‒ время в секундах (по умолчанию 180)

Возобновление работы ‒ `!disallow`
